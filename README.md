# Project 3: Interactive Tic-Tac-Toe Game

## 🎮 Overview
A classic, fully functional Tic-Tac-Toe game built for the web. This project demonstrates complex logic handling in JavaScript, including win-condition checking, draw detection, and dynamic UI updates.

## ✨ Features
- **Two-Player Mode:** Play locally with a friend.
- **Status Indicator:** Real-time updates on whose turn it is and who won.
- **Game Reset:** Quickly restart the match without refreshing the page.
- **Responsive Layout:** Optimized for mobile and desktop screens.

## 🛠️ Tech Stack
- **HTML5:** Game board structure.
- **CSS3:** Custom grid styling and hover effects (using `style4.css`).
- **JavaScript (ES6):** Core game logic and event listeners (using `script2.js`).

## 📁 File Structure
- `index.html` - The main game interface.
- `style4.css` - Visual design for the 3x3 grid and buttons.
- `script2.js` - Logic for turns, winning patterns, and game state.

## 🚀 How to Run
1. Clone this repository.
2. Open `index.html` in your favorite browser.
3. Tap any square to start the game!
